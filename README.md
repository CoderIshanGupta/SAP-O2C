# SAP O2C Simulation Project

## Overview
This project simulates the SAP Order-to-Cash (O2C) process using a web application.

## Features
- Sales Order Simulation (VA01)
- ATP Check
- Delivery Processing (VL01N)
- Goods Issue (VL02N)
- Billing (VF01)
- Payment (F-28)

## Tech Stack
- Python (Flask)
- HTML/CSS
- SAP Concepts (SD, FI, MM)

## How to Run
pip install -r requirements.txt
python app.py

## Purpose
This project validates SAP O2C theoretical concepts through practical implementation.
